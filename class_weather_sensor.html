bool WeatherSensor::getData ( uint32_t  timeout,
uint8_t  flags = 0,
uint8_t  type = 0,
void(*)()  func = NULL 
)

Wait for reception of data or occurrence of timeout. With BRESSER_6_IN_1, data is distributed across two different messages. Reception of entire data is tried if 'complete' is set.

Parameters
timeouttimeout in ms.
flagsDATA_COMPLETE / DATA_TYPE / DATA_ALL_SLOTS
typesensor type (combined with FLAGS==DATA_TYPE)
funcCallback function for each loop iteration. (default: NULL)
Returns
false: Timeout occurred. true: Reception (according to parameter 'complete') successful.
